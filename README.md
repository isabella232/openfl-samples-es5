OpenFL Samples (ES5)
====================

## Get Started

### Clone the repository

```bash
git clone https://github.com/openfl/openfl-samples-es5
```

### Change the directory to one of the samples

```bash
cd openfl-samples-es5/helloworld
```

### Install and start

```bash
npm install
npm start
```
